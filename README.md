<h1>Smart Task Manager 📝</h1>
<h2>Description 📄</h2>
This project is a REST API task manager application with advanced features.
<h2>Features ⚡</h2>
<h3>User Authentication 🔒</h3>
Users can register and log in to access the system securely. Authentication ensures that only authorized users can perform operations.
<h3>Task Operations ✅</h3>
<ul>
  <li><b>Add Tasks ➕:</b> Users can effortlessly add new tasks to the system.</li>
  <li><b>View Tasks 👀:</b> Users can effortlessly add new tasks to the system.</li>
  <li><b>Edit Tasks ✏️:</b> Users have the flexibility to modify task details as needed.</li>
  <li><b>Delete Tasks 🗑️:</b> Unwanted tasks can be removed to keep the task list organized.</li>
  <li><b>Mark as Complete ✔️:</b> Users can mark tasks as complete for effective progress tracking.</li>  
</ul>
<h3>Step Operations ✅</h3>
<ul>
  <li><b>Add Steps ➕:</b> Users can effortlessly add new steps to the system.</li>
  <li><b>View Steps 👀:</b> Users can effortlessly add new steps to the system.</li>
  <li><b>Edit Steps ✏️:</b> Users have the flexibility to modify step details as needed.</li>
  <li><b>Delete Steps 🗑️:</b> Unwanted steps can be removed to keep the step list organized.</li>
  <li><b>Mark as Complete ✔️:</b> Users can mark steps as complete for effective progress tracking.</li>  
</ul>
